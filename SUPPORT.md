## Bug reports and feature requests

Use [GitHub Issues](https://github.com/react-native-nitro-google-signin/google-signin/issues).

Include:

- React Native version
- `react-native-nitro-google-signin` and `react-native-nitro-modules` versions
- **Expo Go vs development build** vs bare React Native
- Platform (Android / iOS) and OS version
- `webClientId: 'autoDetect'` or explicit Web client ID
- Relevant logs (redact tokens and client secrets)

## AI-assisted development

Install the agent skill:

```bash
npx skills add react-native-nitro-google-signin/google-signin -g -y
```

See [For AI agents](https://react-native-nitro-google-signin.github.io/docs/agents/overview).
